## ⬇️ Installation

### Installation from [Crates.io]

```bash
cargo add valuand
```

### Installation from [NPM]

```bash
npm install valuand
bun add valuand
pnpm add valuand
yarn add valuand
```

### Installation from [Pub.dev]

```bash
dart pub add valuand
flutter pub add valuand
```

### Installation from [PyPI]

```bash
pip install -U valuand
uv add valuand
poetry add valuand
pdm add valuand
```

### Installation from [RubyGems]

```bash
gem install valuand
bundle add valuand
```
